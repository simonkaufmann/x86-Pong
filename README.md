# x86-Pong

Pong in Bootsector of x86 PC

## How to use

Make sure **nasm** and **genisoimage** are installed.  

Execute:  
``make all``  

File **boot.iso** will have been created.

Burn ISO image onto CD or add to virtual machine as
Boot Disc (e.g. VirtualBox)
